# Experimental Analysis of Feature Engineering Techniques for Text Classification

## Overview
This project evaluates the impact of feature engineering methods on machine learning model performance for text classification.

## Methods Compared
- Count Vectorization
- TF-IDF

## Models Used
- Logistic Regression
- Multinomial Naive Bayes

## Results
The experiment demonstrates how TF-IDF representation can improve predictive performance by weighting discriminative terms more effectively.

## Tools & Libraries
- Python
- Scikit-learn
- Pandas
- Matplotlib
